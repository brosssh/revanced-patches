To use my patches, I recommend using the awesome [Universal ReVanced Manager](https://github.com/Jman-Github/Universal-ReVanced-Manager/releases/latest).

Simply install the manager and add my bundle as a remote URL using this URL:
`https://raw.githubusercontent.com/Jman-Github/ReVanced-Patch-Bundles/bundles/patch-bundles/brosssh-patch-bundles/brosssh-latest-patches-bundle.json`

Alternatively, you can use the official ReVanced manager by setting `brosssh` as organization in the alternative source section, and `revanced-patches` as the repo name.
